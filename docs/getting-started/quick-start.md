# 快速入门

在 5 分钟内启动并运行 PlotCraft。

## 前置要求

- Node.js 18+
- npm 9+ 或 pnpm 8+
- Git

## 安装

### 1. 克隆仓库

```bash
git clone https://github.com/Agions/PlotCraft.git
cd PlotCraft
```

### 2. 安装依赖

```bash
npm install
```

### 3. 配置 AI 提供商

在项目根目录创建 `.env.local` 文件：

```bash
# AI 提供商 API 密钥
VITE_ALIBABA_API_KEY=***
VITE_OPENAI_API_KEY=***

# 可选：图像生成
VITE_SEEDDREAM_API_KEY=***
VITE_REPLICATE_API_KEY=***
```

### 4. 运行开发服务器

```bash
npm run dev
```

应用将在 `http://localhost:5173` 可用

## 7 步工作流程

```
┌─────────────────────────────────────────────────────────────────┐
│  步骤 1: 导入  →  步骤 2: 生成  →  步骤 3: 故事板               │
│       ↓                   ↓                    ↓               │
│  上传小说/脚本         AI 脚本生成         智能面板设计            │
│                                                                 │
│  步骤 4: 角色 →  步骤 5: 渲染  →  步骤 6: 合成                  │
│       ↓                   ↓                    ↓               │
│  角色设计              场景渲染            动态合成                │
│                                                                 │
│                        步骤 7: 导出                              │
│                            ↓                                    │
│                    视频导出                                       │
└─────────────────────────────────────────────────────────────────┘
```

## 分步指南

### 步骤 1: 导入

导入您的源材料：
- **小说**：上传 TXT 或 Markdown 格式的小说文件
- **脚本**：导入现有的视频脚本
- **提示词**：输入自定义的 AI 提示词

### 步骤 2: 生成

AI 分析您的内容并生成：
- 带场景的结构化脚本
- 带描述的角色列表
- 场景分解

### 步骤 3: 故事板

使用以下内容生成故事板画面：
- 镜头角度（广角、中景、特写）
- 构图建议
- 对话位置

### 步骤 4: 角色

设计角色：
- 外观自定义
- 表情预设
- 语音选择

### 步骤 5: 渲染

生成视觉内容：
- 场景图像
- 角色肖像
- 背景素材

### 步骤 6: 合成

组装您的视频：
- 时间线编辑
- 转场效果
- 音频同步

### 步骤 7: 导出

导出您的最终视频：
- 支持多种格式（MP4、WebM、MOV）
- 质量设置（720p、1080p、4K）
- 字幕嵌入

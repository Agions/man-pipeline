基于 React 18 + TypeScript 5 + Tauri 2.0 构建，提供完整的类型定义和 API 文档，开箱即用。

  • ✓完整 TypeScript 类型覆盖，IDE 自动补全
  • ✓模块化服务架构，易于扩展新 AI 模型

  • ✓Jest + Vitest 双重测试覆盖

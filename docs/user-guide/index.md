# 用户指南

PlotCraft 七步 AI 驱动工作流，将你的故事转化为专业级视频内容。

## 工作流程

1. [导入与分析](./import-analysis) — 上传小说/剧本/提示词
2. [脚本生成](./script-generation) — AI 生成结构化脚本
3. [分镜头设计](./storyboard-design) — 自动生成分镜图
4. [角色设计](./character-design) — AI 角色一致性
5. [渲染与导出](./rendering-export) — 批量渲染和导出
